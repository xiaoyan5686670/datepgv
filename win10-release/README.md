# win10-release

Windows 10 batch installers for datepgv. Use **CRLF** line endings on `.bat` (see [.gitattributes](../.gitattributes)).

See [README.WINDOWS.zh.md](../README.WINDOWS.zh.md) (Chinese) and [README.WINDOWS.md](../README.WINDOWS.md) (English). Script sources are duplicated in [docs/WIN10_RELEASE_CODE.md](../docs/WIN10_RELEASE_CODE.md).
